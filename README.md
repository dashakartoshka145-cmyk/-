Космическая доставка: курьер на Марсе - это мобильная 2D игра в которой игрок выступает оператором марсианского курьерского ровера.
За одну миссию игрок выбирает заказ, оценивает расстояние, проверяет запас энергии, отправляет ровер по маршруту и получает награду за успешную доставку.
основные механики это, проверка расхода энергии, просмотр расстояния до заказа, засчитывание очков, победа и поражение.
основные сцены это, проигрыш, зарядка на станции, выбор заказа, получение очков репутации.
ассеты есть в папке assets.
